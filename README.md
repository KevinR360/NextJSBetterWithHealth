## Getting Started

First, run the build server:

```bash
npm run build
# or
yarn build
```

Now, start the project:

```bash
npm run start
# or
yarn start
```

Case you want run dev: 

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:9045](http://localhost:9045) with your browser to see the result.

Implemented with:

- [x] NextJS;
- [x] React;
- [x] Styled-components;
- [x] axios;
